Test Guru
Программа для тестирования знаний
Краткое описание
Цель проекта - создание приложение для составления тестов, проведение тестирования и оценки знаний.

Выполняется в рамках учебного курса по основам Ruby on Rails школы (Thinkmetica)
Преимущества:
  • Гибкое составление тестов
  • Уровни сложности в заданиях
  • Автоматизированная проверка
  • Групповые политики доступа
